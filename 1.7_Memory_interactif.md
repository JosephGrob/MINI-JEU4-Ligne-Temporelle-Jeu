# Jeu de Mémoire - Réforme

## Introduction

Ce projet propose un jeu de mémoire interactif où les utilisateurs doivent associer des cartes en paires pour découvrir les grandes figures de la Réforme. À chaque paire trouvée, les personnages apparaissent sur le célèbre Mur des Réformateurs, rendant le jeu éducatif et visuellement engageant. L'objectif final est de révéler toutes les figures et de compléter le mur.

---

## Fonctionnalités principales

1. **Structure de la page et style CSS** : créer une interface claire et visuellement engageante. 
2. **Jeu de mémoire interactif** : Cliquer sur les cartes pour les retourner et associer les paires. 
3. **Mur des Réformateurs et Personnages trouvés** : Au fil du jeu, les figures célèbres de la Réforme apparaissent sur le mur. Aussi, Une liste s'enrichit avec les personnages découverts, avec leurs images et noms.
4. **Message de réussite** : Une fois toutes les paires trouvées, un message de félicitations s'affiche.

## Jeu de mémoire interactif 

#### Fonctionnalités principales : 
- Les cartes sont générées dynamiquement et mélangées.
- Les utilisateurs cliquent pour retourner les cartes, et des paires correspondantes sont détectées.

#### Code utilisé : 
1. Les cartes sont générées en associant chaque image à un personnage de la Réforme. Puis, mise en place d'un système qui les mélange à chaque fois que le mini-jeu s'ouvre et mise en place d'un système de retournement des cartes après un laps de temps défini. Aussi, mise en place d'un gestionaire d'événements tactiles permettant à l'utilisateur de cliquer sur les cartes pour les retourer.

```javascript

        const imageMapping = {
            'https://cdn.britannica.com/84/160584-050-68712D6A/Guillaume-Farel.jpg': 'Guillaume Farel',
            'https://upload.wikimedia.org/wikipedia/commons/4/44/Jean_Calvin.png': 'Jean Calvin',
            'https://upload.wikimedia.org/wikipedia/commons/d/d2/Theodore-de-Beze-2.jpg': 'Théodore de Bèze',
            'https://upload.wikimedia.org/wikipedia/commons/a/a7/Portrait_of_John_Knox_%284671577%29.jpg': 'John Knox',
            'https://upload.wikimedia.org/wikipedia/commons/thumb/a/a7/Ulrich_Zwingli.jpg/220px-Ulrich_Zwingli.jpg': 'Ulrich Zwingli',
            'https://regenerationandrepentance.wordpress.com/wp-content/uploads/2015/10/2999922450.jpg': 'Pierre Viret',
            'https://upload.wikimedia.org/wikipedia/commons/5/58/SebastianCastellio.jpg': 'Sebastian Castellio',
            'https://museeprotestant.org/wp-content/uploads/2013/12/0000000044L.jpg': 'Heinrich Bullinger'
        };

        // Création du jeu de cartes en dupliquant les clés de imageMapping (pour former des paires)
        // Mélange des cartes de manière aléatoire
        const cards = [...Object.keys(imageMapping), ...Object.keys(imageMapping)].sort(() => 0.5 - Math.random());
        // Variables pour suivre les cartes retournées et le nombre de paires trouvées
        let flippedCards = [];
        let matchedPairs = 0;
        // Sélection du conteneur pour le jeu de cartes
        const container = d3.select("#game-container");
        // Création des éléments DOM pour chaque carte
        const cardElements = container.selectAll(".card")
            .data(cards)
            .enter()
            .append("div")
            .attr("class", "card")
            .on("click", function(event, d) {
                // Gestionnaire d'événements tactiles pour retourner une carte au clic
                const card = d3.select(this);
                if (!card.classed("flipped") && flippedCards.length < 2) {
                    // Si la carte n'est pas déjà retournée et qu'il y a moins de 2 cartes retournées
                    card.classed("flipped", true); // Marquer la carte comme retournée
                    flippedCards.push(card); // Ajouter la carte à la liste des cartes retournées

                    if (flippedCards.length === 2) {
                        // Si deux cartes sont retournées, vérifier si elles forment une paire
                        checkForMatch();
                    }
                }
            });

        // Ajout des images aux cartes    
        cardElements.append("img")
            .attr("src", d => d)
            .attr("alt", "Card Image");

        // Initialisation du compte à rebours
        let timeRemaining = 30;
        const timerElement = d3.select("#timer");
        const timerContainer = d3.select("#timer-container");

        // Retourner toutes les cartes initialement pour les montrer aux joueurs
        cardElements.classed("flipped", true);

        // Démarrer un intervalle pour gérer le compte à rebours
        const timerInterval = setInterval(() => {
            timeRemaining--;
            timerElement.text(timeRemaining);
            if (timeRemaining <= 0) {
                // Lorsque le temps est écoulé
                clearInterval(timerInterval); // Arrêter le minuteur
                cardElements.classed("flipped", false); // Retourner toutes les cartes face cachée
                d3.select("#message").style("display", "block"); // Afficher le message pour commencer à jouer
                timerContainer.style("display", "none"); // Masquer le conteneur du minuteur
            }
        }, 1000);
```

## Création du mur des réformateurs et Personnages trouvés.

#### Fonctionnalité principale : 
- A chaque paire trouvée dans le memory, le personnage correspondant à cette paire apparait sur le mur des réformateurs uniquement s'il fait partie des personnages présents sur ce mur. Une fois les quatre personnages trouvés, l'image complète du mur permet à l'utilisateur de lui faire découvrir ce monument. Aussi, à droite du memory, un système permet de rséumer tous les personnages trouvés dans le memory en affichant les images des cartes trouvées et le nom des personnages.

#### Fonctionnalité principale : 
- Système de vérification si toutes les paires du memory ont été trouvées et mise à jour du LocalStorage pour indiquer à "index.html" que le jeu est complété. 
- Si c'est le cas : message de réussite.
